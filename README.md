## 📌 Project Overview
This project presents an interactive Customer Churn Analysis Dashboard created using Microsoft Excel. It helps identify key factors influencing customer attrition and provides insights to improve retention.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Slicers (for interactivity)

## 📈 Key Insights
- Customers with month-to-month contracts show higher churn
- Higher monthly charges are associated with increased churn
- Customers with longer tenure are less likely to churn
